% ---- The problem ----

In multi-quadrotor cooperative cable-suspended payload transport, $N$ underactuated vehicles are tethered to a shared rigid payload through flexible cables, forming a system on the configuration manifold $\mathcal{Q} = \SEthree \times (\SEthree \times \Sph^2)^N$ with $6 + 8N$ degrees of freedom. Geometric control methods~\cite{lee2010geometric,sreenath2013geometric,lee2018geometric,wu2014geometric} provide almost-global exponential stability on $\SOthree$ and enable aggressive trajectory tracking with cable-suspended loads, bringing multi-vehicle cable transport close to deployment readiness~\cite{estevez2024review,sun2025agile}. Yet every cable is a single point of mechanical failure. When a cable snaps, three coupled events occur simultaneously: the payload accelerates downward (${\approx}3.3$~m/s$^2$), the freed agent accelerates upward (${\approx}6.5$~m/s$^2$), and every remaining agent experiences an instantaneous load-share jump---a $50\%$ increase for $N{=}3$. These effects propagate on timescales from sub-$10$~ms (rigid-body force transmission) to ${\sim}200$~ms (cable wave propagation), creating a multi-channel fault signature that each surviving agent must absorb without centralized coordination or inter-agent communication.

The difficulty is compounded by the architecture's defining property. The baseline GPAC (Geometric Position and Attitude Control) hierarchy enforces an \emph{N-agnostic constraint}: each agent knows only its own mass, inertia, local sensor measurements, cable tension and direction, and a shared reference trajectory. It knows neither the team size, the payload mass, nor any other agent's state. This constraint eliminates single points of failure and communication dependency, making the architecture scalable---but it also blocks model-based fault detection methods such as multiple-model adaptive estimation (which requires $N{+}1$ hypothesis filters). The tension-channel signal-to-noise ratio for a remote cable break is approximately~$1.3$, indistinguishable from wind perturbations. Cable-fault tolerance in this setting must therefore work without detecting, isolating, or recovering from the fault.

% ---- The gap ----

Despite mature geometric transport theory and extensive UAV fault tolerance work for actuator and sensor failures~\cite{adaika2025fdd,yu2022review}, no published work addresses cable-fault tolerance in cooperative cable-suspended aerial transport. The only adjacent result, Liang~et~al.~\cite{liang2021fault}, assumes known failure identity and centralized reconfiguration---both incompatible with N-agnostic architectures. Zeng~et~al.~\cite{zeng2025decentralized} and Gao~et~al.~\cite{gao2024decentralized} observed implicit resilience to agent loss in learning-based transport but did not formalize the mechanism, and neither addressed cable-suspended underactuated systems.

% ---- The key observation ----

The research objective was shaped by an empirical observation. The predecessor system Tether\_Lift~\cite{hajieghrary2026tetherlift} established that the GPAC hierarchy can coordinate cooperative lift using only local measurements under the N-agnostic constraint. The present paper (Tether\_Grace) asks whether that same architectural property also yields resilience to abrupt cable-topology change.

The answer lies in PID feedback with gravity feedforward. The existing GPAC baseline, evaluated under cable-snap faults with no explicit fault handling, exhibited stable post-fault tracking with payload RMSE of $44$--$49$~cm across $N \in \{3, 5, 7\}$, decreasing with team size despite cascading faults. The mechanism is the N-agnostic PID structure: each agent computes gravity feedforward from its own mass, PID feedback from its own position error, and anti-swing damping from local cable angular velocity. None of these terms depends on $N$, $m_L$, or any other agent's state. When a cable snaps, the surviving agents' tracking errors update instantaneously through the physics, and the PID integral term absorbs the increased load share without adaptation or re-convergence delay. Ablation confirms that disabling cable-tension feedforward changes RMSE by less than $1\%$ (Section~\ref{sec:tension_ablation}).

This transforms the research problem from \emph{how to detect and recover from cable faults} to \emph{whether the emergent fault tolerance can be formalized and quantified}: under what conditions does an N-agnostic controller with local cable-tension feedforward produce bounded tracking degradation across all cable topologies?
Figure~\ref{fig:paper_scope} summarizes this narrowing from the broader Tether\_Lift baseline to the present Tether\_Grace fault-tolerance question and its current evidence boundary.

% ---- The approach ----

This paper addresses that question by identifying the structural property that makes the architecture topology-invariant, deriving operationally useful bounds, and validating them with comprehensive simulation evidence. The replayed full-Drake path uses the GPAC hierarchy with PID-plus-feedforward control. A post-fault geometry redistribution path is additionally described. Adaptive extensions ($\mathcal{L}_1$, concurrent learning) are implemented in the codebase but remain future integration work.

Two structural properties make the PID-plus-gravity-feedforward design attractive. First, PID feedback with gravity feedforward provides the core fault-absorption capability (confirmed by ablation, Section~\ref{sec:tension_ablation}). Second, all controller terms are functions of the agent's own state, so no agent's control law can be invalidated by a topology change it cannot observe.

These properties motivate a \emph{topology-invariance design hypothesis}: if the post-fault disturbance remains inside a fault-inclusive bound and the local thrust margin is adequate, then the tracking bound is governed by controller bandwidth and disturbance size rather than cable identity. Full-Drake multibody results provide strong numerical support: under matched single-fault conditions (identical fault, varying $N$), payload RMSE decreases from $48.8$~cm to $38.4$~cm for $N{=}3$ to $N{=}7$. A thrust-saturation boundary sweep further identifies a sharp phase transition at $f_{\max} \approx 45$~N that coincides with the theoretical post-fault hover minimum, with the default $150$~N providing $3\times$ headroom.

% ---- Contributions ----

\subsection{Contributions}

\begin{enumerate}[label=C\arabic*]
  \item \emph{Topology-invariant error structure and mechanism identification:} A structural analysis (Proposition~1) showing that the N-agnostic PID-plus-feedforward control law produces per-agent error dynamics that are identical across all cable topologies---cable snaps change the disturbance input but not the system matrices. This structural property motivates a topology-invariance hypothesis that is validated by full-Drake simulation. A finite-time transient prediction (${\sim}18$~cm predicted, observed ${\sim}107$~cm; the ${\sim}5.8\times$ ratio is topology-invariant across scenarios) and a closed-form cost ratio $\rho_{\mathrm{FT}} = N/(N{-}k)$ provide design criteria for actuator sizing. Ablation confirms PID feedback with gravity feedforward as the primary mechanism (cable-tension feedforward $< 1\%$ RMSE change).
  \item \emph{Comprehensive multibody evidence:} Full-Drake simulation across $N \in \{3, 5, 7\}$ with $1$--$3$ cascading cable-snap faults under both ideal and sensor-realistic (ESKF + Dryden wind) conditions. Monte Carlo validation ($30$ runs per scenario) with randomized fault timing, wind, cable lengths, and initial conditions. Matched single-fault experiments ($48.8 \to 38.4$~cm), fault-count isolation, trajectory generalization, and baseline comparison provide comprehensive deconfounding.
  \item \emph{Quantitative design framework:} A fault-tolerance cost ratio $\rho_{\mathrm{FT}}$ validated against empirical nominal-vs-fault RMSE ratio, a thrust-margin screening methodology validated by the $f_{\max}$ phase transition at the theoretical minimum, and a tilt-corrected capacity analysis for actuator sizing.
\end{enumerate}
